# Feature List / Roadmap

## Current (v1)
- Google OAuth login
- Browse template gallery (trending/latest, filter by property/type)
- Preview template (video/image)
- Share via WhatsApp, Facebook, Download
- Browse real-estate properties (search + location filter)
- My Studio: Recent & Favorites
- Profile: personal info, company branding, notification preferences

## Future (v2+)
- In-app editor: layers, drag & drop, text, filters, branding overlays
- User media upload (logo, background, music)
- Save custom edits as Drafts, background rendering (FFmpeg/BullMQ)
- Save-to-Studio and Share-with-Team on shared templates
- Real notification delivery (push/email/SMS)
- AI captions & hashtags
